# IoT Home Automation and Monitoring System

This project is based on ESP32 and Blynk platform. It allows users to control and monitor home appliances remotely using a mobile application.

## Features
- Remote control of appliances
- Real-time monitoring
- IoT-based automation

## Components Used
- ESP32
- Relay Module
- Sensors (pir /ir / Ultrasonic if used)
- Blynk App

## Working
The ESP32 connects to WiFi and communicates with the Blynk app. Users can control devices and monitor sensor data in real-time.
